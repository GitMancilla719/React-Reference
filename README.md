# Folder Structure

This follows the "feature folder" approach from Redux [https://redux.js.org/style-guide/#structure-files-as-feature-folders-with-single-file-logic]

/scripts
--index.js
--/app
----store.js
----App.js
--/common (hooks, generic components, utils, etc)
--features
----/sampleFeature
------featureslice.js
------feature.jsx
